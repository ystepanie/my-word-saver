# 나만의 단어장 저장기 (My Own Vocabulary Saver)

웹서핑 중 모르는 단어를 즉시 번역하고 마크다운(`.md`) 형식의 단어장으로 저장해주는 크롬 확장 프로그램입니다.

## 주요 기능
- **간편한 번역**: 드래그한 영단어를 구글 번역 API를 통해 한국어로 즉시 번역합니다.
- **마크다운 저장**: 번역된 단어와 뜻, 날짜를 템플릿에 맞춰 내 컴퓨터의 다운로드 폴더에 저장합니다.
- **AI 학습 연동**: 저장된 단어장 파일을 ChatGPT나 Gemini에게 전달하여 퀴즈 생성 등 학습에 활용할 수 있습니다.

## 설치 및 사용 방법
1. 이 저장소의 파일들을 내 컴퓨터에 다운로드합니다.
2. 크롬 브라우저의 확장 프로그램 관리 페이지(`chrome://extensions/`)로 이동합니다.
3. 오른쪽 상단의 '개발자 모드'를 켭니다.
4. '압축해제된 확장 프로그램을 로드' 버튼을 누르고 폴더를 선택합니다.
5. 웹 사이트에서 단어를 드래그한 뒤 마우스 우클릭 -> **'단어장에 저장하기'**를 클릭하세요!
